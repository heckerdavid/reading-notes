# Loops and iteration

JavaScript loops:

- for
- do...while
- labeled
- break
- continue
- for...in
- for...of

- for loops

    for ([initialExpression]; [conditionExpression]; [incrementExpression])
     statement

- Do...while loops

    do
     statement
    while (condition);

- While
  - a while statement executes as long as the (condition) is True

    while (condition)
      statement

- Dont infinite loop, use break statements

    break;
    break [label];

- Continue

    continue restarts the loop

        continue [label];
